
tes3.saveGame
========================================================

**Parameters:**

- `string`_ **file** ``optional``:
    The filename of the save that will be created, without extension. Defaults to ``quiksave``.

- `string`_ **name** ``optional``:
    The display name of the save. Defaults to ``Quicksave``.

**Returned:**

- **None.**

Saves the game.
